SeismicX Analysis Platform

Advanced deep learning tools for seismic phase detection, association, and analysis

Phase Picker

Detect seismic phases (Pg, Sg, Pn, Sn) from continuous waveform data using state-of-the-art deep learning models. Supports multiple model architectures with automatic visualization.

  • Multiple DL models (PNSN, PhaseNet, EQT)
  • Real-time processing
  • Automatic plotting

Phase Associator

Associate phase picks from multiple stations into coherent earthquake events. Choose from REAL, FastLink, or GaMMA algorithms for optimal results.

  • Three association methods
  • 3D hypocenter location
  • Quality control metrics

Polarity Analyzer

Analyze first-motion polarity of P-waves for focal mechanism determination. Deep learning-based classification with confidence scoring and visualization.

  • Up/Down classification
  • Confidence scoring
  • Waveform visualization
