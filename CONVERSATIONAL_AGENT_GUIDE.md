# 对话式AI助手使用指南

SeismicX现在支持**自然语言对话**来控制地震分析流程！您可以通过对话完成震相检测、关联、极性分析等任务，无需记忆复杂命令。

## 🎯 核心功能

### 1. 智能意图识别
- 理解自然语言指令
- 自动识别您想执行的任务
- 提取关键参数(文件路径、模型名等)

### 2. 多轮对话
- 记住上下文信息
- 支持追问和补充信息
- 连续工作流(拾取→关联→分析)

### 3. 技能自动调用
- 自动选择正确的skill
- 构建并执行命令
- 返回结果和建议

### 4. 交互式引导
- 缺少信息时主动询问
- 提供参数建议
- 确认关键操作

```

```

## 🐛 故障排查

### 问题1: 助手不理解我的意思

**解决**:
- 使用更清晰的表达
- 包含关键词(拾取、关联、极性)
- 查看帮助: "你能做什么"

### 问题2: 总是要求提供文件路径

**解决**:
- 确保路径格式正确
- 使用绝对路径
- 检查文件是否存在

### 问题3: 命令执行失败

**解决**:
- 检查文件权限
- 确认依赖已安装
- 查看详细错误信息

### 问题4: 上下文丢失

**解决**:
- 刷新页面(Web)
- 重启CLI
- 明确指定参数
