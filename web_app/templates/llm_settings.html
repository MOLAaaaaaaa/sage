Help & Tips
Choosing a Provider:
  • Ollama: Best for privacy, works offline, requires installation from ollama.ai
  • OpenAI: High quality models, requires API key from platform.openai.com
  • Anthropic: Great for reasoning, get API key from console.anthropic.com
Recommended Ollama Models:
  • qwen2.5:7b - Good balance of speed and quality (~4GB)
  • llama3.2:3b - Fast and lightweight (~2GB)
  • llama3.1:8b - General purpose model (~4.7GB)
