## 🧠 智能特性

### 1. 意图识别

**关键词匹配**:
```python
'phase_picking': ['拾取', '检测', 'pick', 'detect', '震相']
'phase_association': ['关联', 'association', 'locate', '定位']
'polarity_analysis': ['极性', 'polarity', '初动', 'first motion']
```

**正则表达式**:
```python
r'拾取.*震相'
r'detect.*phase'
r'关联.*震相'
```

### 2. 实体提取

**自动提取**:
- 文件路径: `/data/waveforms`
- 模型名称: `pnsn.v3`, `llama3.2`
- 数值参数: `0.5`, `100`

### 3. 上下文管理

**保持信息**:
```python
class ConversationContext:
    current_task: str          # 当前任务
    task_state: Dict           # 任务状态
    last_results: Dict         # 上次结果
    conversation_history: List # 对话历史
```

### 4. 智能引导

**缺少信息时**:
```
User: 关联震相

Assistant: 我需要更多信息：
- input_file (震相拾取文件路径)
- station_file (台站文件路径)

请提供这些信息。
```

## 📊 技术架构

```
用户输入
    ↓
IntentClassifier (意图识别)
    ↓
提取意图 + 实体
    ↓
SkillExecutor (技能执行)
    ↓
检查参数完整性
    ↓
[缺少信息] → 请求补充
[参数完整] → 构建命令
    ↓
执行命令 (可选)
    ↓
ResponseGenerator (响应生成)
    ↓
返回自然语言回复
    ↓
更新上下文
```

## 🔒 安全性

### 命令执行安全

1. **确认机制**: 执行前询问用户
2. **命令验证**: 检查命令合法性
3. **沙箱执行**: 限制执行权限
4. **日志记录**: 记录所有操作

### 数据安全

1. **本地处理**: 数据不上传
2. **隐私保护**: 不收集对话内容
