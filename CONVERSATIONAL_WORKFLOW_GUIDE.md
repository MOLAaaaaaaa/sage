# 对话式数据分析完整流程指南

## 概述

SeismicX 现在支持通过自然语言对话完成完整的地震数据分析流程：
**数据浏览 → 波形绘制 → 震相检测 → 震相关联 → 极性分析**

所有操作都可以通过简单的对话完成，无需记忆复杂的命令。

## 常见错误和解决

### 错误 1: 意图识别不正确

**问题：** 说"检测 /data 目录"被识别为震相检测而不是浏览

**原因：** "检测"关键词匹配震相检测

**解决：** 使用更明确的表达
```
✗ "检测 /data 目录"
✓ "查看 /data 目录有哪些文件"
✓ "浏览 /data 目录"
```

### 错误 2: 找不到文件

**问题：** "文件不存在"错误

**原因：** 
- 路径拼写错误
- 使用了相对路径但当前目录不对
- 文件确实不存在

**解决：**
```
✓ 使用绝对路径
✓ 先用 "查看" 确认文件存在
✓ 检查路径拼写
```

### 错误 3: 绘图失败

**问题：** "绘制波形时出错"

**可能原因：**
- 文件格式不支持
- 文件损坏
- ObsPy 未安装

**解决：**
```bash
# 检查依赖
pip install obspy matplotlib

# 验证文件格式
obspy-read /path/to/file
```

## Web 聊天界面使用技巧

### 快速操作按钮

Web 界面提供了快捷按钮：
- 📁 **浏览数据** - 快速浏览
- 📊 **绘制波形** - 快速绘图
- 🎯 **拾取震相** - 快速检测
- 🔗 **关联事件** - 快速关联
- 📐 **极性分析** - 快速分析

### 图像交互

- 波形图自动显示在聊天窗口
- 点击"下载图片"保存 PNG
- 右键图像可以"在新标签页打开"查看原图

### 清空对话

- 点击"清空对话"按钮清除历史
- 刷新页面也会重置对话
- 建议定期清空以保持流畅

## API 集成

如果您想在自己的应用中集成对话功能：

```python
from conversational_agent import get_agent

agent = get_agent()

# 处理用户消息
result = agent.process_message("查看 /data 目录")

# 获取响应
print(result['response'])

# 检查是否有图像
if result.get('image_url'):
    print(f"Image available at: {result['image_url']}")

# 检查是否需要执行命令
if result['action'] == 'execute_command':
    cmd = result['data']['command']
    # 执行命令...
```

### 组合命令

```

```

```
3. 对整个目录执行检测
4. 统一关联所有拾取
5. 批量分析极性
```

## 故障排除清单

遇到问题时检查：

- [ ] 路径是否正确（使用绝对路径）
- [ ] 文件是否存在（先用"查看"确认）
- [ ] 依赖是否安装（obspy, matplotlib）
- [ ] 意图识别是否正确（检查置信度）
- [ ] 文件格式是否支持（.mseed, .sac, .seed）
- [ ] 权限是否足够（读取文件权限）
