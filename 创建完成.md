### 2. CLI统一接口

**文件**: `seismic_cli.py`

**命令**:
```bash

python seismic_cli.py pick -i <输入> -o <输出> -m <模型> -d <设备>

# 震相关联

# 极性分析
python seismic_cli.py polarity -i <拾取文件> -w <波形目录> -o <输出文件>
```

## 📝 使用示例

### 示例1: 完整工作流
```bash
# 1. 震相检测
python seismic_cli.py pick \
    -i /data/waveforms \
    -o results/picks \
    -m pnsn/pickers/pnsn.v3.jit \
    -d cuda:0 \
    --enable-polarity

# 2. 震相关联
python seismic_cli.py associate \
    -i results/picks.txt \
    -o results/events.txt \
    -s data/stations.txt \
    -m real

# 3. 极性分析
python seismic_cli.py polarity \
    -i results/picks.txt \
    -w /data/waveforms \
    -o results/polarity.txt \
    --min-confidence 0.7
```

### 示例2: Web界面操作
1. 启动服务: `cd web_app && python app.py`
2. 打开浏览器: http://localhost:5000
3. 点击"Phase Picker"卡片
4. 填写参数表单
5. 点击"Start Phase Picking"
6. 查看任务状态和日志
7. 下载结果文件

## 🔧 配置说明

### 震相检测配置
编辑 `pnsn/config/picker.py`:
```python
prob = 0.3              # 置信度阈值
nmslen = 1000           # NMS窗口
polar = True            # 启用极性检测
ifplot = True           # 启用绘图
```

### 关联算法配置
编辑 `pnsn/config/fastlink.py`:
```python
ngrid = 20              # 网格大小
nps = 6                 # 最小震相数
np = 3                  # 最小P震相数
ns = 2                  # 最小S震相数
```
